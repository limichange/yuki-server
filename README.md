## install

### npm
```shell
$ npm i -g sequelize
$ npm i -g nodemon
```

### yarn
```shell
$ yarn global add sequelize
$ yarn global add nodemon
```

## develop

### run server
```shell
$ nodemon index.js
```

### migrate
```shell
$ ./migration.sh
```
